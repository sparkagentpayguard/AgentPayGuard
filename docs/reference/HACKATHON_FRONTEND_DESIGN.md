# 黑客松获奖项目前端设计研究（索引）

**完整版**：研究型长文已迁至 **[docs/archived/HACKATHON_FRONTEND_DESIGN.md](../archived/HACKATHON_FRONTEND_DESIGN.md)**。

**设计原则摘要**（摘自完整版）：
- 科技感优先：深色主题、霓虹色、发光效果
- 即时反馈：交互 0.2–0.5s 视觉响应
- 信息密度高：充分展示链上数据、避免混乱
- 交互优雅：平滑过渡、无突兀跳转
- 信任可视化：通过视觉明确传达系统状态

**提炼版**：日常开发参考见 [FRONTEND_DESIGN_REFERENCE.md](../guides/FRONTEND_DESIGN_REFERENCE.md)（配色/动画/布局建议）。
